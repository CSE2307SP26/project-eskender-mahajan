## User stories

1. A bank customer should be able to deposit into an existing account. (Shook)
2. A bank customer should be able to withdraw from an account. (Beamlak)
3. A bank customer should be able to check their account balance. (Beamlak)
4. A bank customer should be able to view their transaction history for an account. (Mohini)
5. A bank customer should be able to create an additional account with the bank. (Mohini)
6. A customer should be able to log in using a pin number. (Mohini)
7. A bank administrator should be able to view customer accounts in the bank and interact with them. (Mohini)
8. A customer should be able to to make a savings and a checking account (Beamlak)
9. A saving account and checking account should be able to apply different interest rates. (Beamlak)
10. A customer should be able to apply for large transfers of money between accounts, however, a Bank admin has to approve them. (Mohini)
11. A customer should be able to reset their pin number by answering questions about their account. (Mohini)
12. A bank administrator should be able to view and deny mortgage applications. (Beamlak)
13. A customer should be able to apply for a mortgage. (Beamlak)
